## Overview | 仓库简介

This repository records my learning process for Java and Object-Oriented Programming.

It contains not only Java code written in class and during practice, but also:

- summaries of important knowledge points
- worksheet answers and explanation notes
- my own confusion points and corrected understanding
- small exercises used to improve class design and OOP thinking

This repository serves as both a coding workspace and a revision notebook.

这个仓库记录了我学习 Java 和面向对象编程（OOP）的过程。

它不仅包含课堂和练习中写的 Java 代码，也包含：

- 课程核心知识点总结
- weeksheet / worksheet 答案与解析
- 我自己的疑问点、误区和修正后的理解
- 用来训练类设计和 OOP 思维的小练习

因此，这个仓库不仅是代码仓库，也是我自己的复习笔记和学习档案。

## Purpose | 建立这个仓库的目的

The goal of this repository is not only to store answers.

I use it to:

1. practise Java programming
2. review OOP concepts
3. organise worksheet solutions
4. write down questions I had during learning
5. turn confusing topics into clear notes for revision

Rather than keeping only final answers, I want this repository to reflect my actual learning process: what I learned, what I misunderstood at first, and how I corrected my understanding.

这个仓库的目的，不只是保存答案。

我使用它来：

1. 练习 Java 编程
2. 复习 OOP 核心概念
3. 整理 weeksheet / worksheet 答案
4. 记录学习过程中的疑问点
5. 把原本模糊的知识点整理成以后可复习的清晰笔记

相比只保存“最终答案”，我更希望这个仓库能真实反映我的学习过程：  
我学到了什么、最开始误解了什么、后来又是怎么纠正理解的。
